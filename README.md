# CMSC6950Project

<h1>Description</h1>

SunPy is an open-source Python library for Solar Physics data analysis and visualization.


<h1>Installation</h1>

<ul>
  <li>Anaconda</li>
  <li>SunPy:<br> conda config --append channels conda-forge<br>conda install sunpy</li>
</ul>

<ul>
  <li>Sunpy</li>
  <li>Astropy</li>
  <li>FIDO</li>
  <li>matplotlib</li>
</ul>

<h1>To Use</h1>
<ul>
